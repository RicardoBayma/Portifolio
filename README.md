# 💻 Meu Portfólio

Este é o repositório do meu site do portfólio. Desenvolvi ultilizando Html, JavaScript e CSS.

---

## 🚀 Tecnologias que utilizei:

O projeto foi construído sem o uso de frameworks externos, focando na performance e semântica do código:

- **HTML5:** Estruturação semântica de componentes para melhor acessibilidade e SEO.
- **CSS3:** Estilização moderna baseada em variáveis CSS, além de layouts estruturados com **Flexbox** e **CSS Grid**.
- **JavaScript (Vanilla):** Lógica integrada para manipulação do DOM e ativação do menu responsivo (hambúrguer) em dispositivos móveis.

---
